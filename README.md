# Aplikasi-Sistem-Manajemen-Kos-Berbasis-Desktop
Sistem Manajemen Kos adalah aplikasi desktop berbasis Python (PyQt6) dan SQLite untuk mengelola kamar, penghuni, dan pembayaran. Tersedia dashboard admin, pencatatan tagihan, laporan ekspor Excel, serta login terpisah admin dan user. Ringan, cepat, aman, dan mudah digunakan. Cocok menggantikan pembukuan manual.
